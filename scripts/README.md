# Scripts

This folder is reserved for utility scripts. Development commands are in root `package.json`:

- `npm run dev:all` – backend + frontend
- `npm run docker:up` / `docker compose up` – app stack (URLs from `backend/.env`; see [docs/setup/DOCKER.md](../docs/setup/DOCKER.md))
- `npm run setup` – install dependencies

See [README.md](../README.md) and [docs/setup/DOCKER.md](../docs/setup/DOCKER.md) for full Quick Start.
