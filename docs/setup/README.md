# Setup Documentation

This directory contains all setup and installation guides for the Stock Insight App.

## Quick Navigation

- **[Quick Start](./QUICK_START.md)** - Fastest way to get started (PostgreSQL setup + servers)
- **[Environment Setup](./ENV_SETUP.md)** - Detailed environment variables configuration
- **[Run Servers](./RUN_SERVERS.md)** - Comprehensive guide for running development servers
- **[Architecture Setup](./ARCHITECTURE_SETUP.md)** - Architecture overview and structure
- **[Start Development](./START_DEVELOPMENT.md)** - Complete development environment setup

## Setup Order

For new developers, follow this order:

1. **Quick Start** - If you just want to get running quickly
2. **Architecture Setup** - Understand the project structure
3. **Environment Setup** - Configure all environment variables
4. **Start Development** - Complete setup guide
5. **Run Servers** - Detailed server management

## Common Setup Tasks

### First Time Setup
1. Install dependencies (see [Start Development](./START_DEVELOPMENT.md))
2. Set up environment variables (see [Environment Setup](./ENV_SETUP.md))
3. Set up database (see [Run Servers](./RUN_SERVERS.md))
4. Start servers (see [Run Servers](./RUN_SERVERS.md))

### Daily Development
1. Start PostgreSQL (if not running)
2. Start backend server
3. Start frontend server

See [Run Servers](./RUN_SERVERS.md) for detailed commands.
